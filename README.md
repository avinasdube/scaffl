# Scaffl

AI-powered software architecture generator

Scaffl transforms natural language requirements into:

- Entities
- Relationships
- Database schemas
- API endpoints
- ER diagrams
- Starter project boilerplates


## Example

```
    User can login.
    Admin can ban users.
    Posts support comments.
```

## Output

```
    {
        "entities": [],
        "relationships": [],
        "apis": []
    }
```
